# Embedding Content

:::note Speedrun Docs Guide

You should remove this page when making your docs.

:::

## Images

1. Add the image to the `static` folder at the root directory.
2. Add `![Title of image](<path to file>)`.

Example:

`![Speedrun Docs Logo](/img/run-man.svg)`
![Speedrun Docs Logo](/img/run-man.svg)

:::tip Speedrun Docs Logo

Feel free to modify and customise the logo to your desired game! An example can be seen in the [Heavenly Bodies Docs](https://ewanlyon.github.io/HeavenlyBodies-SpeedrunDocs/).

:::

## Videos

Videos are crucial in teaching speedruns. We have some barebones iframe wrappers included by default in the template. Here's how to link to different places:

:::info MDX

When embedding extra content like videos, we recommend changing the file extension to `.mdx` (Markdown Extension) as it is no longer standard Markdown. Docusaurus will still parse regular Markdown (`.md`) files as MDX though.

:::

### YouTube

```JSX
<YouTubeVideo videoId="DI8vHDrvJGc" startTime={943} />
```

| Property   | Type   | Description                                        |
|------------|--------|----------------------------------------------------|
| videoId    | string | The YouTube video's ID                             |
| startTime  | number | *Optional.* The number of seconds to start a video |

<YouTubeVideo videoId="DI8vHDrvJGc" startTime={943} />

### Twitch

```JSX
<TwitchVideo videoId="LuckyDirtyPheasantWOOP-KmnZnto7H0NjjaQM" />
```

| Property   | Type   | Description                                                                                                                                      |
|------------|--------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| videoId    | string | The YouTube video's ID                                                                                                                           |
| startTime  | number | *Optional.* The number of seconds to start a video                                                                                               |
| parents    | string | *Optional.* If you need to specify the website URL. Learn more from [Twitch's Documentation](https://dev.twitch.tv/docs/embed/video-and-clips/). |

<TwitchVideo videoId="LuckyDirtyPheasantWOOP-KmnZnto7H0NjjaQM" />
